# examples

## 「dot.setup_fil.OS名.conf」ファイル

setup_fil.sh 用の「変数定義ファイル」のサンプルファイルです。  
いずれかのファイルを「.setup_fil.OS名.conf」というファイル名でホームディレクトリにコピー後、
必要に応じて内容を編集してください。
